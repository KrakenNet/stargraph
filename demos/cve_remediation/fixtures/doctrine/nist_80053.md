# NIST 800-53r5 Subset (offline fixture)

## Access Control (AC)

### AC-2 Account Management
Establish, activate, modify, review, disable, remove accounts. Maps to CWE-287 (improper authentication).

### AC-3 Access Enforcement
Enforce approved authorizations. Maps to CWE-285 (improper authorization).

## System and Information Integrity (SI)

### SI-2 Flaw Remediation
Identify, report, correct system flaws. Includes patch management, vulnerability scanning. Aligns with the cve_remediation pipeline directly.

### SI-7 Software, Firmware, Information Integrity
Verify integrity of software/firmware via checksums or signatures. Maps to CWE-345 (insufficient verification of data authenticity).
